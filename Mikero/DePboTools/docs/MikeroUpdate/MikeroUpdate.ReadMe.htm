Mikero's Tools Updater version 1.xx

see readme general and fixes.txt

This gui maintains a mirror of files available from dev-heaven on your pc and can't possibly be simpler to use.

For each file, either your copy is the latest 'n greatest or, you don't have the latest version (if any at all).

Double clicking on any file will cause it to be downloaded and installed, or, re-installed

first time install

The first time you activate this gui, no files will be 'installed' because the depot has not been built. This will be a small annoyance to those of you who already have the correct versions. The gui cannot know what you downloaded in the past. But, should you wish to, you can move your current installer.exe's (if you still have them them), to C:\Program Files (x86)\Mikero\DePboTools\downloads to avoid some tedium.
